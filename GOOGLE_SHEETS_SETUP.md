# Настройка Google Sheets интеграции

## Шаг 1: Создать Google Sheets таблицу

1. Перейдите на https://sheets.google.com
2. Нажмите **"Создать"** → **"Таблица"**
3. Назовите её: **"RetailCRM"**
4. Создайте листы с названиями:
   - Клиенты
   - Товары
   - Продажи
   - Сотрудники
   - История операций
   - Отчеты

## Шаг 2: Настроить заголовки листов

### Лист "Клиенты"
```
ID | Имя | Email | Телефон | Город | Дата добавления
```

### Лист "Товары"
```
ID | Название | Цена USD | Цена UAH | Запас | Минимум | Статус
```

### Лист "Продажи"
```
ID | Дата | Товар | Количество | Цена USD | Продавец | Статус
```

### Лист "Сотрудники"
```
ID | Имя | Email | Должность | Оклад USD | Комиссия % | Продажи USD | Комиссия USD | Всего USD
```

### Лист "История операций"
```
Дата/Время | Пользователь | Операция | Детали | Статус
```

### Лист "Отчеты"
```
Метрика | Значение
```

## Шаг 3: Получить Google API ключ

1. Перейдите на https://console.cloud.google.com
2. Создайте новый проект: **"RetailCRM Bot"**
3. Включите API:
   - Google Sheets API
   - Google Drive API
4. Создайте Service Account:
   - Перейдите в **"Учетные записи сервисов"**
   - Нажмите **"Создать учетную запись сервиса"**
   - Назовите: **"retail-crm-bot"**
   - Нажмите **"Создать"**
5. Создайте ключ:
   - Откройте созданную учетную запись
   - Перейдите в **"Ключи"**
   - Нажмите **"Добавить ключ"** → **"Создать новый ключ"**
   - Выберите **JSON**
   - Нажмите **"Создать"**
   - Скачается JSON файл

## Шаг 4: Поделиться таблицей с Service Account

1. Откройте JSON ключ, найдите **"client_email"**
2. Откройте вашу Google Sheets таблицу
3. Нажмите **"Поделиться"**
4. Вставьте email из JSON ключа
5. Дайте права **"Редактор"**
6. Нажмите **"Поделиться"**

## Шаг 5: Получить ID таблицы

1. Откройте вашу Google Sheets таблицу
2. В URL найдите ID между `/d/` и `/edit`:
   ```
   https://docs.google.com/spreadsheets/d/[ID]/edit
   ```
3. Скопируйте этот ID

## Шаг 6: Настроить Railway

1. Откройте Railway проект
2. Перейдите в **Variables**
3. Добавьте две переменные:

### GOOGLE_SHEETS_ID
```
[Ваш ID таблицы из шага 5]
```

### GOOGLE_SHEETS_CREDENTIALS
```
[Содержимое JSON ключа из шага 3]
```

4. Нажмите **Save**
5. Railway автоматически перезагрузит бота

## Шаг 7: Проверить синхронизацию

1. Откройте @crm0424bot в Telegram
2. Добавьте клиента через бот
3. Откройте Google Sheets таблицу
4. Проверьте, что клиент появился в листе "Клиенты"

## Готово! ✅

Теперь все данные из бота автоматически записываются в Google Sheets!

## Примечания

- Бот читает данные из Google Sheets при запуске
- Все новые данные записываются в таблицу в реальном времени
- История всех операций сохраняется в листе "История операций"
- Отчеты обновляются автоматически
