# RetailCRM Telegram Bot 🤖

Повнофункціональна CRM система для управління магазином через Telegram.

## Функціональність

### 👥 Управління клієнтами
- Додавання нових клієнтів
- Перегляд списку всіх клієнтів
- Синхронізація для всіх 4 співробітників

### 🛍️ Управління продажами
- Реєстрація нових продаж
- Вибір товару та кількості
- Автоматичне зменшення запасу

### 📦 Управління складом
- Перегляд статусу всіх товарів
- Індикатор низького запасу (⚠️)
- 10 товарів (iPhone, iPad, AirPods, тощо)

### 💰 Розрахунок зарплати
- Автоматичний розрахунок комісії
- Для всіх 4 співробітників
- Оклад + комісія = Всього

### 📊 Звіти
- Всього продаж
- Загальний дохід
- Активні клієнти
- Товари на складі

### 📝 Історія операцій
- Запис всіх дій
- Час, користувач, операція
- Останні 5 операцій

## Встановлення

### Локально

1. Клонуйте репозиторій:
```bash
git clone https://github.com/your-username/telegram-crm-bot.git
cd telegram-crm-bot
```

2. Встановіть залежності:
```bash
pip install -r requirements.txt
```

3. Встановіть BOT_TOKEN:
```bash
export BOT_TOKEN="your-bot-token-here"
```

4. Запустіть бота:
```bash
python bot.py
```

### На Railway

1. Перейдіть на [railway.app](https://railway.app)
2. Натисніть "New Project"
3. Виберіть "Deploy from GitHub"
4. Виберіть цей репозиторій
5. Додайте змінну оточення `BOT_TOKEN` з вашим токеном
6. Натисніть "Deploy"

## Змінні оточення

- `BOT_TOKEN` - Токен Telegram бота (обов'язково)
- `SUPABASE_URL` - URL Supabase (опціонально)
- `SUPABASE_KEY` - Ключ Supabase (опціонально)

## Як користуватися

1. Знайдіть бота в Telegram за його ім'ям
2. Натисніть `/start`
3. Оберіть дію з меню
4. Всі дані синхронізуються для всіх 4 співробітників
